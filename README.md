# NeighborWise AI

NeighborWise AI is an AI-powered neighborhood decision-support platform that helps users compare areas using rent prices, school performance indicators, school boundaries, safety levels, GIS maps, Power BI dashboards, and a virtual assistant.

## Project Goal

The goal of this project is to help renters, families, newcomers, and residents make better housing decisions by combining reliable data sources, GIS mapping, dashboard analytics, and AI recommendations.

## Main Features

- Rent price analysis by neighborhood, bedroom type, and property type
- School information and school performance indicators
- School boundary and neighborhood GIS map layers
- Safety level and neighborhood safety score
- Power BI dashboard with interactive filters and maps
- AI virtual assistant for natural-language neighborhood recommendations

## Technology Stack

- Python
- Pandas
- SQL / PostgreSQL
- Power BI
- GIS / GeoJSON / shapefiles
- LangChain
- LangGraph
- Streamlit
- Data Lake
- Data Warehouse
